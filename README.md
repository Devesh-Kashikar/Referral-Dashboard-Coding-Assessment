# Go Business Referral Dashboard

A responsive React.js web application for managing referrals, tracking earnings, and monitoring partner activity.

## 🔗 Live Demo

referral-dashboard-coding-assessment-git-main-devesh15.vercel.app

## 📂 GitHub Repository

https://github.com/Devesh-Kashikar/Referral-Dashboard-Coding-Assessment

---

## 🚀 Features

- User authentication with JWT
- Protected routes
- Dashboard overview metrics
- Service summary section
- Referral sharing with copy functionality
- Search referrals
- Sorting
- Pagination
- Referral details page
- Logout functionality
- Responsive design

---

## 🛠 Tech Stack

- React.js
- React Router DOM
- Axios
- JS Cookie

---

## 📦 Installation

```bash
npm install
npm run dev
```
